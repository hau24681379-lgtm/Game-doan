# GAME STORE PROJECT (SUPABASE)

Dự án này đã được cấu hình sử dụng cơ sở dữ liệu Supabase của bạn.

## Các lệnh quan trọng

### 1. Tại thư mục `backend/`
- `npm run migrate:latest`: Khởi tạo bảng trên Supabase.
- `npm run seed:run`: Nạp tài khoản mẫu (admin/player).
- `npm run dev`: Chạy server (Port 3000).

### 2. Tại thư mục `frontend/`
- `npm run dev`: Chạy giao diện (Port 5173).
